# Phase 3 — Final Code Directory

## Location for C++ Source Files

Place your completed C++ source files in this folder.

### Expected Files

| File Name          | Description                              |
|--------------------|------------------------------------------|
| `main.cpp`         | Main program entry point                 |
| `routes.h`         | Header file for route data structures    |
| `routes.cpp`       | Route processing functions               |

---

> **Reminder:** Your program must:
> - Compile without errors
> - Read data from `../data/routes_input.txt`
> - Write results to `../data/routes_output.txt`
> - Use at least one array and one vector
> - Use Bubble Sort OR Selection Sort
> - Use linear search
