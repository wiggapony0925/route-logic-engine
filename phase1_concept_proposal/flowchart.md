2. **Open input file** (`routes_input.txt`)

4. **Process data** (calculate times, distances)
5. **Sort data** (Bubble Sort or Selection Sort)
6. **Search** for a specific route number (linear search)
7. **Write results** to output file (`routes_output.txt`)
8. **Program End**
