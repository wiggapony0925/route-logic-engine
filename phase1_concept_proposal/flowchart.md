# Phase 1 - Flowchart

## Program Flowchart

The flowchart shows the menu system for the Bus Route Manager:

1. Program starts.
2. Read route data from `routes_input.txt`.
3. Store the route numbers, route names, distances, and travel times.
4. Display the main menu.
5. Get the user's menu choice.
6. If the user chooses option 1, display all bus routes.
7. If the user chooses option 2, ask for a route number and use linear search to find it.
8. If the route is found, display the route information. If it is not found, display a route not found message.
9. If the user chooses option 3, sort the routes by travel time using Bubble Sort and display the sorted routes.
10. If the user chooses option 4, compare the route distances or times and display the shortest and longest route.
11. If the user chooses option 5, write a summary report to `routes_output.txt` and save the file.
12. If the user chooses option 6, end the program.
13. After options 1 through 5 finish, the program returns to the main menu.

### Required Steps to Show in Flowchart

1. **Program Start**
2. **Open input file** (`routes_input.txt`)
3. **Store route data** including route numbers, route names, distances, and travel times
4. **Display main menu** and get the user's choice
5. **Display routes** when the user chooses option 1
6. **Search** for a specific route number using linear search when the user chooses option 2
7. **Sort routes** by travel time using Bubble Sort when the user chooses option 3
8. **Find shortest and longest route** when the user chooses option 4
9. **Write summary report** to `routes_output.txt` when the user chooses option 5
10. **Program End** when the user chooses option 6
