# Phase 1 - Output Plan

## What results will your program generate?

The program will create an output report with the route list sorted by travel time. The report will also include the shortest route, the longest route, and a search result for one route number.

## Output File Name

File name: `routes_output.txt`

## Output File Format Example

```
Bus Route Manager - Summary Report

Routes Sorted by Duration:
189 RooseveltIsland_Shuttle 20:10 15 3.2
137 BatteryPark_Shuttle 11:30 16 3.5
194 CivicCenter_Shuttle 21:00 16 3.6
101 TimesSquare_Shuttle 05:30 18 4.2

Shortest Route: 189 RooseveltIsland_Shuttle, 15 minutes
Longest Route: 129 RockawayBeach_Express, 64 minutes

Search Result for Route 101:
101 TimesSquare_Shuttle 05:30 18 4.2
```

## Summary Statistics Included

- All routes sorted by travel duration
- Shortest route by travel time
- Longest route by travel time
- Search result for a requested route number
