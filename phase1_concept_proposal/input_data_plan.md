# Phase 1 - Input Data Plan

## What data will your program read?

File name = `routes_input.txt`

- Route number (integer)
- Route name (string)
- Departure time (string)
- Travel time in minutes (integer)
- Route distance in kilometers (number)


## File Format Example

File name: routes_input.txt

```
RouteNumber RouteName DepartureTime Duration_min Distance_km
101 TimesSquare_Shuttle 05:30 18 4.2
102 GrandCentral_Local 05:40 22 5.8
103 UnionSquare_Loop 05:50 24 6.4
104 WallStreet_Express 06:00 28 8.7
```

## Data Fields

- `RouteNumber`: number used to identify the bus route
- `RouteName`: name of the bus route
- `DepartureTime`: time the route starts
- `Duration_min`: travel time in minutes
- `Distance_km`: route distance in kilometers
