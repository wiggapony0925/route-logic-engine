# LLM Conversation Log

## Instructions

Place your exported AI conversation PDF in this folder.

### Required File

| File Name                  | Description                        |
|----------------------------|------------------------------------|
| `ai_conversation_log.pdf`  | Exported PDF of your full AI chat  |

---

### Your PDF must include

- All questions you asked the AI
- All responses you received
- Notes on how you used the responses
